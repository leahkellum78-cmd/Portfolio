# Leah Kellum — Multimedia Portfolio

A single-page portfolio site showcasing video production, web design, and photography work.

**Live Site:** [leahkellum78-cmd.github.io](https://github.com/leahkellum78-cmd/leahkellum78-cmd)

## About

This portfolio was built as part of a multimedia creative course at Texas Tech University. It highlights my primary focus in video editing and VFX, alongside web design and photography work completed throughout the program.
